Comandos básicos

  1. configurando o git
    • git config --global user.name "seu nome" configurando seu nome no git.
    • git config --global user.email "seu email" configurando seu email no git.
    • git config --global core.editor "nome do seu editor" configurando o nome do seu editor (ex: netbeans).
    • Para ver as configurações
    • git config --list lista todas as configurações.

  2. git status - para ver os arquivos modificados e os arquivos "trakeados".
  3. git add [nome do arquivo com exensão] - para "trakear um arquivo", preparar para commit.
  4. git add * - para "trakear todos arquivos" no diretório em que se encontra.
  5. git commit -m - commitar (guardar no repositório local) e deixar uma mensagem de descrição.
  6. git log - ver todos "commits" efetuados.
  7. git branch - ver em qual branch eu estou.
  8. git push - enviar commit para o repositório remoto.
  9. git push origin master - enviar commit para o repositório remoto na branch master.
