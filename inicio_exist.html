Início - projeto existente

  1. Com o git intalado.
  2. Abra o terminal do GIT e navegue até a pasta do projeto.
  3. Execute o comando git init para criar um repositório local.
  4. Logado em sua conta no github.
  5. Crie (inicie) um novo repositório.
  6. Coloque apenas o nome do projeto e uma descrição, desmarque todas as outras opções, o projeto deve ser criado em branco.
  7. Acesse seu projeto no github, clique em "clone or download" e copie o link que avi aparecer.
  8. No terminal do GIT excute git remote add origin [link copiado]
  9. execute o comando git remote deverá aparecer a palavra "origin".
  10. Para verificar se você está no projeto correto execute git remote -v que irá aparecer o link do projeto.
  11. Caso o remote esteja errado execute o comando git remote remove origin
  12. E para enviar seu projeto para o repositório remoto execute o comando abaixo:
  13. git push -u origin master
  14. aqui pulamos para a parte de comandos básicos.
