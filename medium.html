Comandos intermediários

  • Clonando repositório online.
    1. Entre no projeto no github e copie o link de clonagem.
    2. Abra o terminal do GIT e navegue até a pasta onde deseja clonar, execute o comando abaixo.
    3. git clone [link copiado] - clona o repositório online criando uma pasta com o nome do projeto.
  • Revertendo modificações
    1. git reset --soft [id do commit] volta ao commit escolhido mantendo os outros commits realizados trakeados, ou seja, prontos para commitar novamente.
    2. git reset --mixed [id do commit] volta ao commit escolhido mantendo os outros commits realizados mas não os deixa trakeados, sendo assim, terá que ser efetuado o comando git add *.
    3. git reset --hard [id do commit] exclui os commits (e arquivos gerados nesse commit) que foram feitos depois do commit escolhido (id commit).
  • Trabalhando com Branches
    1. git branch ver em qual branch eu estou e lista todos os branches existentes.
    2. git branch [nome do branch] cria um novo branch.
    3. git checkout [nome do branch] mudo para o brach escolhido.
  • Diferença entre arquivos
    1. git diff ver quais linhas foram afetadas.
    2. git diff --name-only ver apenas uma lista com o nome dos arquivos que tiveram seu conteúdo alterado.
    3. git diff [nome do arquivo com extensão] ver apenas as modificações no arquivo escolhido.
    4. git checkout HEAD -- [nome do arquivo com extensão] volta o arquivo escolhido à sua situação inicial, ou seja, ao estado anterior a suas modificações.
  • Mesclando arquivos
    1. git branch --no-merged ver branch's não mescladas.
    2. git checkout master vou para a minha branch master.
    3. git merge [nome da branch q será mesclada à master] -m "menssagem" efetua a mesclagem de branch's unificando os códigos.
    4. git log --oneline --decorate --parents ver todo o log de commit com seus cod pais, melhorando assim a forma de ver as mesclagens.
